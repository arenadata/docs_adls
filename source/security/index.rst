Руководство по работе с Apache Ranger
=====================================


В документе приведены настройки функций безопасности Hadoop, авторизация в кластере, установка Apache Ranger и создание HDFS Policy.

Документ может быть полезен администраторам, программистам, разработчикам и сотрудникам подразделений информационных технологий, осуществляющих внедрение кластера.

.. important:: Перед установкой убедитесь, что версия бандла Arenadata Hadoop не ниже чем 2.1.3.0

.. important:: Контактная информация службы поддержки -- e-mail: info@arenadata.io
